# LLM/VLM Deployment Best Practices

This repository contains guidelines and best practices for deploying Large Language Models (LLMs) and Vision-Language Models (VLMs). It covers optimization strategies for both **low latency** and **high throughput** performance, with a focus on practical implementation.

## Contents
- **latency-guidelines.md**: Strategies for optimizing model performance for low-latency, real-time applications.
- **throughput-guidelines.md**: Strategies for maximizing model throughput to handle a high volume of requests.
